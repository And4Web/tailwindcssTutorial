😍 Hello Tailwind! 😍

1. Height and width

Height and Width properties can be used in Tailwind CSS something like this:

h-32 w-1/3
h-16 w-1/3
h-screen w-32
h-32 w-full
h-vh w-2/3

For more Height and Width Properties, refer official docs: Tailwind Sizing: Heights and Widths

2. Border and Colors

Border and Colors properties can be used in Tailwind CSS something like this:

border-2 border-purple-900 rounded
border-4 border-double border-yellow-500 rounded-lg
border-8 border-dotted border-blue-500
border-b-4 border-t-4 border-green-500 rounded-b-full
border-b-4 border-t-4 border-green-500 rounded-full
border-4 border-dashed border-blue-500 rounded-tr-xl rounded-bl-lg

For more border and Colors Properties, refer official docs: Tailwind Borders > Tailwind Colors

3. Margins and Paddings

Margins and Paddings properties can be used in Tailwind CSS something like this:

mt-3 p-4
mt-3 pt-4
mb-3 pb-4
my-3 pl-7
mx-3 pr-7

For more Margin, Padding and space-between Properties, refer official docs: Tailwind Margin > Tailwind Padding > Tailwind Space Between
