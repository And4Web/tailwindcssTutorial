Hello Tailwind!

1. Height and width

Height and Width properties can be used in Tailwind CSS something like this:

h-32 w-1/3
h-16 w-1/3
h-screen w-32
h-32 w-full
h-vh w-2/3

For more Height and Width Properties refer official docs: Tailwind Sizing: Heights and Widths
